# Publishing support & privacy URLs

These pages live in this folder and are ready for **GitHub Pages**.

## Expected URLs (after deploy)

| Purpose | URL |
|---------|-----|
| **Support URL** (App Store Connect) | `https://waynebuckley1980.github.io/tradekeep/` |
| **Privacy Policy URL** | `https://waynebuckley1980.github.io/tradekeep/privacy.html` |

Repo: **WayneBuckley1980/tradekeep** on GitHub Pages from the `/docs` folder.

## One-time setup

1. Create a GitHub repo named `tradekeep` (or push this project to GitHub).
2. In the repo on GitHub: **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** → folder **/docs**
5. Save — the site is live in 1–2 minutes.

## App Store Connect

Paste the URLs above into:

- **App Information → Support URL**
- **App Privacy → Privacy Policy URL** (or App Information, depending on ASC layout)
